# FileHandlingPractice

A C# Console Application created to practice file handling and basic data persistence.

## Topics Covered

* Reading and writing text files using `System.IO`
* `File.WriteAllText()`
* `File.WriteAllLines()`
* `File.ReadAllText()`
* `File.ReadAllLines()`
* `File.AppendAllText()`
* `File.Exists()`
* `Directory.CreateDirectory()`
* `Path.Combine()`
* `Path.GetFileName()`
* `Path.GetExtension()`
* `Path.GetFileNameWithoutExtension()`

## Features

* Create a list of students.
* Save students to a text file.
* Load students from a text file.
* Append new data to a file.
* Check if a file exists before reading it.
* Create directories and files programmatically.
* Extract file name and extension using the `Path` class.

## Technologies

* C#
* .NET
* System.IO
